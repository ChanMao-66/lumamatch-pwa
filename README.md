# LumaMatch

LumaMatch 是一個本機優先（local-first）的 AI 攝影風格匹配 App。使用者選擇參考照片與自己的原圖，App 會匹配色彩、明暗與氛圍，提供智慧構圖，並以原圖寬高輸出。

## 第一版已完成

- 參考照片與原圖雙上傳
- 六种无需参考图的原创摄影风格范本
- 风景／食品场景识别与 AI 风格推荐
- “惊艳增强”局部对比、通透度与高光色彩重塑
- 色彩均值、對比與明暗分佈匹配
- 自然膚色保護
- 可調式風格強度
- 明暗、對比與冷暖微調
- 原圖／效果拖曳比較
- 視覺重心分析與三種構圖
- 90° 旋转、精细旋转与自动水平校正
- AI 推荐修图、自然磨皮、人物提亮与细节增强
- 中文文字／语音描述修图
- JPG、PNG 原始寬高輸出
- PWA 安裝與離線使用

照片處理全部在瀏覽器或手機裝置內完成，不會上傳到伺服器。

构图模式会维持原图的输出宽度与高度；选择裁切构图时，画面会重新取景并缩放到原尺寸，因此“像素尺寸”不变，但裁切掉的真实细节无法凭空恢复。生成式扩图会留到后续版本。

## GitHub Pages

GitHub Pages 从 `main` 分支的 `docs/` 目录发布。部署成功后，用手机打开 Pages 地址即可使用，并可通过浏览器选单“加入主画面”安装为 PWA。

## 直接預覽

可使用任何靜態伺服器開啟 `docs/`。例如：

```bash
npm install
npm run serve
```

## 后续路线

目前为可运行 MVP，采用快速的本地统计色彩匹配。后续可接入语义分割模型、天空／人物蒙版、局部光影重建、RAW 解码与生成式扩图。功能与介面正式确认后，再统一整理为 Android／iOS 上架项目。
